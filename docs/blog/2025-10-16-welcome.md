## What is Agent Kernel?

Agent Kernel is a framework-agnostic runtime for AI agents that lets you:

- Build agents once, run anywhere
- Switch between frameworks without rewriting code
- Deploy to AWS Lambda, ECS, or run locally
- Test agents interactively with built-in CLI
- Expose agents via REST API, MCP, or A2A protocols
