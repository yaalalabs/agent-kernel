# AWS Bedrock Guardrails

:::info Coming Soon
AWS Bedrock Guardrails support is planned for a future release of Agent Kernel. This page outlines the planned functionality and configuration.
:::

## Overview

AWS Bedrock Guardrails will provide native integration with Amazon Bedrock's guardrails service, enabling:

- **Content Filtering**: Block harmful or inappropriate content
- **PII Detection and Redaction**: Identify and redact sensitive information
- **Topic-based Blocking**: Control conversations based on topics
- **Prompt Attack Detection**: Detect and block adversarial prompts
- **Contextual Grounding**: Ensure responses are grounded in provided context
